<h1>Инструкции по запуску Задания 1</h1>

<ol>
    <li><strong>Убедитесь, что у вас установлен Python:</strong>
        <pre>python --version</pre>
    </li>
    <li><strong>Установите библиотеку <code>dnspython</code>:</strong>
        <pre>pip install dnspython</pre>
    </li>
    <li><strong>Сохраните скрипт:</strong>
        <p>Скопируйте код в файл с именем <code>domens.py</code>.</p>
    </li>
    <li><strong>Запустите скрипт из командной строки:</strong>
        <pre>python domens.py</pre>
    </li>
</ol>

<h1>Инструкции по запуску Задания 2</h1>

<ol>
    <li><strong>Создайте текстовый файл:</strong>
        <p>Создайте файл <code>message.txt</code> и добавьте в него необходимый текст для отправки.</p>
    </li>
    <li><strong>Установите библиотеку <code>requests</code> (если она еще не установлена):</strong>
        <pre>pip install requests</pre>
    </li>
    <li><strong>Настройте скрипт:</strong>
        <p>Откройте скрипт и замените <code>YOUR_TELEGRAM_BOT_TOKEN</code> и <code>YOUR_CHAT_ID</code> на ваши значения.</p>
    </li>
    <li><strong>Сохраните скрипт:</strong>
        <p>Скопируйте код в файл с именем <code>telegram.py</code>.</p>
    </li>
    <li><strong>Запустите скрипт из командной строки:</strong>
        <pre>python telegram.py</pre>
    </li>
</ol>
